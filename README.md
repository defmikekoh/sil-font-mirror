# SIL Font Mirror

Full (non-subsetted) WOFF2 web fonts from [SIL International](https://www.sil.org/), mirrored here for use as a CDN source.

Google Fonts hosts these fonts but serves subsetted versions that break composite glyphs (e.g., IPA tone letters, combining diacritical marks). These are the complete fonts from SIL's official downloads.

## Fonts

| Font | Version | Type | Source |
|------|---------|------|--------|
| [Andika](https://software.sil.org/andika/) | 7.000 | Sans-serif | [Download](https://software.sil.org/andika/download/) |
